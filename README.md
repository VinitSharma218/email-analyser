### Email Analyzer 

This is a simple yet smart email analyzer where user can put the header of the email which he/she wants to check. After that, on the basis of authentication results, as well the content of the email, AI will give the proper results of the anlyzed email.

## How to run the project?
- cd email-analyser (or any folder where you cloned this project)
- npm install
- npm run dev

Remember that you have to create a .env file also where you have to put your own GEMINI_API_KEY for flash-2.5 model to leverage the Gemini features.
