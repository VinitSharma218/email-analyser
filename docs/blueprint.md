# **App Name**: Header Analyzer

## Core Features:

- Header/EML Input: Provide a text area or file upload for the user to input the email header or the whole .eml file.
- Header Analysis: Analyze the email header to extract information like DMARC, SPF, DKIM, etc.
- Security Summary: Provide a summary of the possibility of spam or spoofing using AI as a tool, based on the header analysis.
- Extracted Links, Domains, and IP Addresses: Display a separate list of all the links, domains, and IP addresses found within the email header.
- VirusTotal Integration: Provide VirusTotal links for each IP address and domain for further analysis.

## Style Guidelines:

- Primary color: Sky blue (#87CEEB) to reflect trust and clarity.
- Background color: Light gray (#F0F8FF) for a clean and professional look.
- Accent color: Soft green (#90EE90) for positive indicators and highlighting key information.
- Body and headline font: 'Inter', a sans-serif font, providing a modern, neutral, and readable appearance.
- Use simple, recognizable icons to represent different types of extracted data and security indicators.
- Organize the analyzed information into clear, distinct sections for easy navigation and understanding.
- Use subtle transitions and animations to enhance user interaction and provide feedback.
